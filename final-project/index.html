Welcome to my Final Project

This project is the culmination of my web development bootcamp, where I applied everything I learned—from structuring websites with HTML and CSS to deploying live on Netlify. It includes a fully functional multi-page site (Home, About, Contact, and Donate) featuring a custom logo, responsive navigation, social media links, and a contact form powered by Netlify Forms. I also integrated a secure donation page using Stripe, allowing visitors to support my work directly. The site showcases embedded media, a Google Map, and interactive elements that reflect best practices in modern web design.
